# CV Interactivo - Juan Antonio Neila
Este proyecto contiene mi CV profesional en formato web, optimizado para lectura digital e impresión.

## Tecnologías utilizadas
* HTML5 / CSS3 (Diseño Responsive)
* GitHub Pages (Hosting)
* Git (Control de versiones)
* SSH para despliegue seguro

## Características
* **Botón de impresión:** Genera un PDF optimizado mediante CSS `@media print`.
* **Despliegue automático:** Cualquier cambio en la rama `main` se publica al instante.
